# استبيان العزلة الإلكترونية

نظام استبيان إلكتروني متكامل لدراسة **"آثار العزلة الإلكترونية ودور الخدمة الاجتماعية في التخفيف من أثارها السلبية لدى الشباب الجامعي"**

## المميزات

- ✅ **تسجيل البيانات الأولية** (النوع، السن، الكلية، الفرقة، استخدام الإنترنت)
- ✅ **6 أقسام للاستبيان**:
  - الهدف من استخدام الإنترنت
  - واقع تفشي ظاهرة العزلة الإلكترونية
  - تأثير العزلة الإلكترونية على التحصيل الدراسي
  - تأثير العزلة الإلكترونية على العلاقات الاجتماعية
  - تأثير العزلة الإلكترونية على الحالة النفسية
  - المقترحات للتخفيف من الآثار السلبية
- ✅ **خيارات إجابات واضحة** (نعم - إلى حد ما - لا)
- ✅ **تحليل فوري للنتائج** مع استخراج النسب المئوية
- ✅ **رسوم بيانية** توضح توزيع الإجابات
- ✅ **لوحة تحكم شاملة** لعرض الإحصائيات والنتائج
- ✅ **إدارة الأسئلة** (إضافة، تعديل، حذف)

## المتطلبات

- Python 3.10+
- Django 5.0+
- matplotlib
- Pillow

## التثبيت

1. تثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

2. تشغيل migrations:
```bash
python manage.py migrate
```

3. إنشاء superuser:
```bash
python manage.py createsuperuser
```

4. إنشاء أسئلة الاستبيان:
```bash
python create_survey_questions.py
```

5. إنشاء بيانات تجريبية (اختياري):
```bash
python create_sample_data.py
```

6. تشغيل الخادم:
```bash
python manage.py runserver
```

7. فتح المتصفح على:
```
http://127.0.0.1:8000/
```

## النماذج (Models)

### Participant (المشارك)
- `name`: الاسم (اختياري)
- `gender`: النوع (ذكر/أنثى)
- `age`: السن
- `college`: الكلية
- `year`: الفرقة الدراسية
- `primary_website`: أهم المواقع المستخدمة
- `computer_skill`: مهارة استخدام الحاسوب
- `daily_hours`: عدد الساعات اليومية

### Question (السؤال)
- `text`: نص السؤال
- `category`: التصنيف (6 تصنيفات)
- `order`: الترتيب
- `is_active`: حالة النشاط

### Answer (الإجابة)
- `participant`: المشارك
- `question`: السؤال
- `answer`: الإجابة (yes/somewhat/no)
